# 🌦️ Bale Weather Bot

A simple and lightweight weather bot for the **Bale Messenger**.

The bot provides current weather information for Iranian cities using the free **Open-Meteo API**.

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Current temperature
- 💧 Humidity
- 🌬️ Wind speed
- ☁️ Weather conditions
- ⚡ Fast responses
- 🇮🇷 Designed for Bale Messenger

## 🛠️ Technologies

- Python
- Bale Bot API
- Open-Meteo API
- Requests

## Installation

Clone the repository:

```bash
https://github.com/arian-it/bale-weather-bot.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the bot:

```bash
python main.py
```

## Configuration

Replace your Bale Bot Token inside the project:

```python
TOKEN = "YOUR_BOT_TOKEN"
```

Weather data is provided by:
